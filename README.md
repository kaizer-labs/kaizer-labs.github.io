# Kaizer Charania 3D Portfolio

A React, TypeScript, GSAP, and React Three Fiber portfolio based on the interaction model from [akashrmalhotra/3d-portfolio](https://github.com/akashrmalhotra/3d-portfolio), adapted with Kaizer-specific content, links, and case-study visuals.

## Scripts

- `npm run dev` starts the local Vite dev server.
- `npm run build` type-checks and creates a production bundle.
- `npm run preview` serves the production build locally.
- `npm run lint` runs ESLint.
